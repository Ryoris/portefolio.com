Création de ce Site Web

Programmation du site

En effet, le site web sur lequel vous lisez ce texte est un projet en soi ! J’avais pour objectif de partager mes expériences à travers un site web simple d’utilisation. J'ai créé le squelette du site avec des fichiers HTML pour chaque page. Chaque fichier contenait les éléments de base comme les en-têtes, les sections de contenu, et les pieds de page.

Ensuite, j'ai ajouté des fichiers CSS pour styliser les pages. J'ai défini les styles pour les différentes sections, les polices, les couleurs, et la disposition générale du site afin de rendre le site attrayant et cohérent visuellement. Quelques fonctionnalités dynamiques, comme des carrousels d'images ou des slides interactifs, ont été codé en JavaScript. Cela a permis d'améliorer l'expérience utilisateur à travers une bonne interactivité.

Utilisation de Git et GitHub

J'ai utilisé Git pour le contrôle de version, ce qui m'a permis de suivre les modifications apportées au site et de revenir à des versions précédentes si nécessaire. J'ai créé un dépôt Git local où je faisais des commit régulier de mes changements. Pour rendre le site accessible en ligne, j'ai utilisé GitHub Pages. J'ai créé un dépôt sur GitHub et poussé mon code vers ce dépôt. Ensuite, j'ai configuré GitHub Pages pour qu'il serve le contenu du dépôt. Note : cela m'a permis de publier le site gratuitement !

Cependant ce n'est pas fini, après la mise en ligne j'ai continué à améliorer le site en fonction des retours et des besoins. À chaque modification, je mettais à jour le dépôt GitHub, ce qui actualisait automatiquement le site en ligne grâce à GitHub Pages. Cette expérience m'a donné une base solide pour continuer à explorer et à développer des projets web plus complexes à l'avenir.

Technologies utilisées : HTML, CSS, Javascript, Github
